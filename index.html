Curso de Modulação

Seja bem-vindo ao Curso de Modulação! Este treinamento é de extrema importância para aprender e relembrar o modo correto de modular na rádio.

Atenção: Este não é um curso opcional. Quem quiser subir sua patente deverá possuir os cursos específicos para cada nível hierárquico.

Estamos disponíveis para dúvidas, não tenha medo de perguntar!

01 Código Q

  • QAP – Na escuta.
  • QRR – Pedido de reforço.
  • QTH – Localização.
  • QRA – Nome e patente.
  • QRL – Estou ocupado.
  • QRQ – Transmita mais rápido.
  • QRS – Transmita mais devagar.
  • QRU – Ocorrência.
  • QTX – Saindo de serviço.
  • QSV / VTR – Viatura.
  • QSL – Entendido.
  • QSO – Comunicar algo importante.
  • QRX – Prioridade (Silenciar rádio).
  • QTA – Abortar.
  • TKS – Obrigado.
  • QSM – Repita a mensagem.
  • QRV – À disposição.
  • QSJ – Dinheiro Sujo.

02 Regras de Modulação

  • Não usar palavras inadequadas (xingamentos, “ah”, “ééé...”, “aqui”, “lá”, “ali”).
  • Sempre prezar por uma modulação breve e rápida; ouça antes de falar.
  • Conversas paralelas apenas via /pd.
  • Mantenha a calma mesmo sob pressão; controle suas emoções.
  • Ao pedir QRR, informe sempre o seu QRA.
  • Comando de QRX exige silêncio imediato na rádio.
  • Não atropele a modulação de outros oficiais.
  • Modulação em VTR pelo P2 (GTM pode ser o piloto).
  • Em perseguição, a primária modula. Se houver QTA, a secundária assume.

03 Código de Patrulha

Código 0: ➥ Iniciando patrulhamento.
Código 1: ➥ Intensidade baixa (trânsito).
Código 2: ➥ Média (Drogas/Mascarados).
Código 3: ➥ Uso não letal / Intensidade alta.
Código 4: ➥ Situação contida / sob controle.
Código 5: ➥ Fogo liberado (Atire para matar).
Código 6: ➥ Perímetro / Investigar (360º).
Código 7: ➥ QRR MÁXIMO (FORÇA TÁTICA).
